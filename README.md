# Example Secure NestJS REST API with Keycloak

## Description

The example to test how [Nest](https://github.com/nestjs/nest) framework integrate with Keycloak OAuth 2.0.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
